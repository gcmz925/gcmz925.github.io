RPM (Red Hat/Fedora) Debian/Ubuntu 等效命令 说明
rpm -ivh pkg.rpm dpkg -i pkg.deb 安装本地包
rpm -e pkg dpkg -r pkg 移除包
rpm -qa dpkg -l 列出所有已安装包
rpm -q pkg `dpkg -l grep pkg` 检查是否安装
rpm -qf /path/to/file dpkg -S /path/to/file 查询文件属于哪个包
yum install pkg apt-get install pkg 从仓库安装
yum remove pkg apt-get remove pkg 卸载包
